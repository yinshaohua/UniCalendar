## 农历日期显示

### 月视图日格中的农历位置

| Option | Description | Selected |
|--------|-------------|----------|
| 日期数字下方小字（推荐） | 公历日期下方显示一行小字农历信息，与 Apple Calendar 中文版风格一致 | |
| 日期数字右侧标注 | 农历信息显示在公历日期右边，同行排列 | ✓ |
| 悬浮提示 | 鼠标悬浮时才显示农历信息，tooltip 方式，不占用日格空间 | |

**User's choice:** 日期数字右侧标注
**Notes:** 无

### 显示内容优先级

| Option | Description | Selected |
|--------|-------------|----------|
| 优先节日/节气，其次农历日（推荐） | 有传统节日时显示节日名，有节气时显示节气名，其它日子显示农历日期 | ✓ |
| 始终显示农历日期 | 只显示农历日期，节日/节气不替换显示 | |
| 显示两行 | 农历日期 + 节日/节气分开两行显示 | |

**User's choice:** 优先节日/节气，其次农历日
**Notes:** 无

### 周/日视图是否显示农历

| Option | Description | Selected |
|--------|-------------|----------|
| 显示（推荐） | 周视图日期 header 和日视图标题都追加农历信息 | |
| 不显示 | 只在月视图显示农历，周/日视图保持简洁 | ✓ |

**User's choice:** 不显示
**Notes:** 无

### 额外用户补充
**User提出:** 月视图标题栏左上角当前月份增加农历月份显示，考虑显示长度增加如果影响右侧功能按钮位置就调整按钮位置。

## 数据与依赖策略

### 农历计算方案

| Option | Description | Selected |
|--------|-------------|----------|
| 第三方库 lunar-js/tyme4ts（推荐） | 成熟的农历库，包含历法转换+节气+传统节日，省开发时间但增加 bundle 体积 | ✓ |
| 自实现轻量版 | 基于寿星万年历算法自写，bundle 最小但开发量大 | |
| Claude 决定 | 让 Claude 在 research 阶段评估后决定 | |

**User's choice:** 第三方库
**Notes:** 无

### 节假日数据来源

| Option | Description | Selected |
|--------|-------------|----------|
| 硬编码当年数据（推荐） | 插件内置当前年度放假/调休日期表，每年更新插件时更新数据 | |
| 在线接口获取 | 从公开 API 或 GitHub 数据源获取假日数据，自动更新但需网络访问 | ✓ |
| 两者结合 | 内置当年数据作为默认，同时提供在线更新能力作为备选 | |

**User's choice:** 在线接口获取
**Notes:** 无

### 设置开关

| Option | Description | Selected |
|--------|-------------|----------|
| 默认开启，可关闭（推荐） | 农历和节假日默认显示，用户可在设置中关闭 | ✓ |
| 默认关闭，可开启 | 默认不显示，需要的用户手动开启 | |
| 始终显示 | 不提供开关 | |

**User's choice:** 默认开启，可关闭
**Notes:** 无

## 额外用户补充

**今天日期样式调整：** 月视图中今天日期不再使用不同背景色，改为只通过日期数字样式（加粗、颜色或圆圈等）区分。背景色差异留给节假日标记使用，避免多种背景色造成视觉杂乱。
