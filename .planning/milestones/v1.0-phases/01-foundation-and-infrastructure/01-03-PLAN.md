    **Class `UniCalendarSettingsTab extends PluginSettingTab`:**
    - `plugin: UniCalendarPlugin`
    - Constructor: `super(app, plugin); this.plugin = plugin;`

    **`display()` method** (called by Obsidian when settings tab opens):

    ```
    const { containerEl } = this;
    containerEl.empty();
    ```

    **Section 1: General Settings**
    - `containerEl.createEl('h2', { text: 'General' });`

    - Sync interval dropdown:
      ```
      new Setting(containerEl)
        .setName('Sync interval')
        .setDesc('How often to automatically sync calendar events')
        .addDropdown(dropdown => dropdown
          .addOptions({ '5': '5 minutes', '15': '15 minutes', '30': '30 minutes', '60': '1 hour' })
          .setValue(String(this.plugin.settings.syncInterval))
          .onChange(async (value) => {
            this.plugin.settings.syncInterval = Number(value);
            await this.plugin.saveSettings();
          }));
      ```

    - Default view dropdown:
      ```
      new Setting(containerEl)
        .setName('Default view')
        .setDesc('Calendar view to show when opening the plugin')
        .addDropdown(dropdown => dropdown
          .addOptions({ 'month': 'Month', 'week': 'Week', 'day': 'Day' })
          .setValue(this.plugin.settings.defaultView)
          .onChange(async (value) => {
            this.plugin.settings.defaultView = value as 'month' | 'week' | 'day';
            await this.plugin.saveSettings();
          }));
      ```

    **Section 2: Calendar Sources**
    - `containerEl.createEl('h2', { text: 'Calendar Sources' });`

    - If `this.plugin.settings.sources.length === 0`: show `containerEl.createEl('p', { text: 'No calendar sources configured. Add one below.', cls: 'setting-item-description' });`

    - For each source in `this.plugin.settings.sources`:
      - Create card div: `const card = containerEl.createDiv({ cls: 'uni-calendar-source-card' });`
      - Left side info div: `const info = card.createDiv({ cls: 'uni-calendar-source-info' });`
        - Color dot: `info.createSpan({ cls: 'uni-calendar-color-dot', attr: { style: 'background:' + source.color } });`
        - Source details div:
          - Name: `detailsDiv.createEl('span', { text: source.name, cls: 'uni-calendar-source-name' });`
          - Type badge: `detailsDiv.createEl('span', { text: source.type.toUpperCase(), cls: 'uni-calendar-source-type' });`
          - Status: enabled/disabled text
      - Right side buttons via Setting:
        ```
        new Setting(card)
          .addExtraButton(btn => btn.setIcon('pencil').setTooltip('Edit').onClick(() => {
            new EditSourceModal(this.app, this.plugin, source, () => this.display()).open();
          }))
          .addExtraButton(btn => btn.setIcon('trash').setTooltip('Delete').onClick(async () => {
            this.plugin.settings.sources = this.plugin.settings.sources.filter(s => s.id !== source.id);
            await this.plugin.saveSettings();
            this.display();
          }));
        ```

    - Add source button at bottom:
      ```
      new Setting(containerEl)
        .addButton(btn => btn
          .setButtonText('Add calendar source')
          .setCta()
          .onClick(() => {
            new AddSourceModal(this.app, this.plugin, () => this.display()).open();
          }));
      ```


    **Class `AddSourceModal extends Modal`:**
    - Step 1: Type selection. Show 3 buttons: "Google Calendar", "CalDAV", "ICS Feed". Each sets selectedType and proceeds to step 2.
    - Step 2: Source configuration form based on type.
      - All types get: Name (text input, required), Color (color picker, pre-filled with `getNextColor(this.plugin.settings.sources)`), Enabled toggle (default true)
      - Type `google`: CalendarId text input (placeholder: "primary")
      - Type `caldav`: Server URL (text), Username (text), Password (text input with type password -- use `inputEl.type = 'password'` on the text component), Calendar path (text, optional)
      - Type `ics`: Feed URL (text, placeholder: "https://...")
    - On save: Generate UUID via `crypto.randomUUID()` (available in modern browsers and Obsidian runtime). Create CalendarSource object. Push to `plugin.settings.sources`. Call `plugin.saveSettings()`. Close modal. Call refresh callback to re-render settings.
    - On cancel: Close modal.

    **Class `EditSourceModal extends Modal`:**
    - Similar to AddSourceModal step 2, but pre-filled with existing source values.
    - Type field is read-only (shown as text, not changeable).
    - On save: Update source in-place in `plugin.settings.sources` array (find by id). Call `plugin.saveSettings()`. Close modal. Call refresh callback.

    IMPORTANT implementation details:
    - Use `crypto.randomUUID()` for source ID generation (no external UUID library needed)
    - CalDAV password field: after creating the Setting with `.addText()`, access the input element and set `inputEl.type = 'password'` for masking
    - Color picker: use `.addColorPicker()` from Setting API. Pre-fill with `getNextColor()`.
    - All modals should have a title set via `this.titleEl.setText('Add Calendar Source')` or similar.
    - The `display()` method calls `containerEl.empty()` then rebuilds everything -- this is the standard Obsidian pattern for dynamic settings.
